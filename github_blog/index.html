blog文章计划

需要制定一些计划去规划文章的分类、编辑、发布等，目的是不让文章在数量增大的情况下管理变的混乱，管理混乱就容易将当前工程变烂。

Tip
当前的这些规划只针对我个人的需求，不具备通用性。

编辑器选择

  • vscode
  • typora

通常我会两个一起使用，typora用做编辑文章，vscode用于测试site页面文章展示及页面代码修改。
